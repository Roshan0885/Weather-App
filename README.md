# Weather App  

## Description  
A simple weather application that fetches live data from the OpenWeatherMap API.  
It displays temperature, humidity, wind speed, and weather icons in a clean UI.  

## Tech Stack  
- HTML  
- CSS  
- JavaScript  
- OpenWeatherMap API
